# Nikcode test task

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## API manual testing using Swagger

You can manually test the API in your browser, by running the app and following this url: http://localhost:3000/docs

## Run Jest Unit Tests

```bash
# unit tests
$ npm run test
```
